람다식 (Lambda Expression)

위에서 작성한 isEven(number)함수를 람다식으로 작성하면 아래와 같다.

람다식을 map 또는 filter와 같은 구조에 가져다 사용할 수 있다.

맵 (Map)

위에서 만든 람다식으로 0부터 10의 숫자들이 각각 짝수인지 홀수인지 확인해보자. 짝수면 True 홀수면 False이다.

map에서 직접 람다식을 작성해도 된다.

필터링 (Filter)

map이 각 원소에 해당 람다식을 적용시킨 것이라면, filter는 람다식 조건에 부합하는 값들만 추려서 반환한다.
